Box Model:

  • margin
    outer layer; clears area around border; no background color, transparent
  • border
    around padding & content; inherited from color property of box
  • padding
    clears area around content; affected by background color
  • content
    image/text; "nucleus" - other layers protect this

Setting Box Sizes:

*{
box-sizing: border-box;
}
**sets size of box so it won't change (margin not included in the size)

drawback: must add browser-specific prefixes:
  • webkit
  • moz
  • ms

Here's the CSS code:
*{
-webkit-box-sizing: border-box;
-moz-box-sizing: border-box;
-ms-box-sizing: border-box;
}

box size in % instead of pix:
affects screen size
size changes, depending how big screen is or how encompassing the box is

if set in pixels, won't change size at all no matter what

max width: box will resize to accommodate the screen size, but not larger than max width
